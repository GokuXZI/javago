# JavaGo

JavaGo is a beginner-friendly Java compile and run tool for Termux and Linux.

It allows students to compile and run Java programs using a single command.

## Features
- Automatic Java installation
- Compile and run in one step
- Clear error messages
- Helpful tips for beginners

## Usage
javago Hello.java

## Example
javago examples/Hello.java
